7가지의 타입

1. 숫자(number)

정수(int)란? - 소수점이 없는 수

실수(float)란? - 소수점이 있는 수 (부동소수점)

2. 문자(charactor)

"문자"는 기호(글자,숫자,문장 부호) 또는 비출력 "제어"이다. UTF-8은 가장 보편적인 문자 세트이며 유명한 인간 언어들의 문자를 포함한다.

3. 문자열 (string)

String 전역 객체는 문자열(문자의 나열)의 생성자 입니다.

4. Boolean,bool : 참(true), 거짓(false)

- 첫 번째 매개변수로서 전달한 값은 필요한 경우 불리언 값으로 변환됩니다.

- True,False 딱 두가지 값 밖에 없다.

- 사람이 핸들링이 가능하다.

- 이것과 저것, 이쪽 아니면 저쪽

- False라고 해서 나쁜 것이 아님(그냥 방향만 다른것 뿐)

5. 묶음, Object, 객체

Object 클래스는 JavaScript의 데이터 유형 중 하나를 나타냅니다. 다양한 키 모음 및 더 복잡한 엔티티들을 저장하는 데 사용됩니다.

JavaScript의 거의 모든 객체는 Object의 인스턴스입니다.

객체 예시 : 가방, 필통, 305호, C++, 추상화

항목이름 : Key

항목값 : Value

항목 : field,entity,property

6. 객체이긴한데 '순서'필요할때

항목 순서 : index

항목값 : value

배열

7. Function(기능) - 만들고 싶은 것을 가능하게 해줌

함수(상자, 기능을 넣는 상자)

함수를 짠다 (기능을 만든다)
